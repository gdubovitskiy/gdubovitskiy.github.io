Обо мне

Прошел путь от аналитики до руководителя Data Engineering-командой в Ozon. Мой аналитический бэкграунд помогает выстраивать эффективные ETL-процессы, которые действительно решают бизнес-задачи.

Специализируюсь на построении эффективной инфраструктуры для обработки данных (batching, streaming), оптимизации их хранения и потребления.

  • Разрабатываю data-пайплайны (Airflow, Spark, Kafka) и витрины данных для аналитики.

  • Внедряю DataOps-практики: CI/CD, мониторинг (Grafana), стандарты Python кода (Poetry, Ruff, Mypy).

  • Выстраиваю Data Governance (Data Quality, Data Contracts, DataHub) для прозрачности и качества данных.

  • Собираю data решения на основе ClickHouse, Vertica, Kafka, S3/MinIO и HDFS.

  • Отобран в Python Experts Committee компании (10 участников из 5000+ сотрудников) - разрабатываю Python стандарты компании в направлении DA/DE.

Работаю на стыке инженерии, аналитики и продуктовой разработки. Умею объяснять сложное просто и находить баланс между производительностью и бизнес-потребностями.
